# METID – Website Informasi Meteorologi
METID merupakan website berbasis web yang dikembangkan sebagai proyek tugas kuliah dengan tujuan menyediakan informasi meteorologi secara interaktif, informatif, dan mudah dipahami. Website ini dirancang untuk membantu pengguna dalam mengakses berbagai informasi cuaca dan iklim secara visual melalui tampilan yang modern dan responsif.Proyek ini mengintegrasikan konsep meteorologi dengan teknologi web development untuk menampilkan data dan visualisasi sebagai media pembelajaran, website ini juga diharapkan dapat meningkatkan kesadaran masyarakat terhadap pentingnya informasi cuaca dan iklim dalam aktivitas sehari-hari.

## Features
- Tampilan website modern dan responsive
- Informasi cuaca dan meteorologi
- Materi terkait klasifikasi iklim
- Dashboard sederhana dan user-friendly

## Purpose
Project ini dibuat untuk memenuhi tugas perkuliahan sekaligus mengembangkan kemampuan dalam:
- Web Development
- Data Visualization
- Implementasi konsep meteorologi dalam teknologi digital
- Kolaborasi tim dalam pengembangan proyek

## Team
Dikembangkan oleh mahasiswa sebagai bagian dari proyek pembelajaran dan pengembangan teknologi di bidang meteorologi.
